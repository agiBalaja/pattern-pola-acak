Pattern Lock 4D | Input Pola ? 8 Hasil Acak

Cara Penggunaan

1. Gambar pola (drag/geser) pada Pattern Input (pertama).
2. Pola yang digambar akan menjadi seed untuk menghasilkan 8 angka 4D acak.
3. Hasil acak (8 pattern di bawah) otomatis diperbarui dan read-only (tidak bisa digambar ulang).
4. Setiap kali menggambar pola baru di input, hasil acak akan berubah.
